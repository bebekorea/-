# 01. 요구사항 정의서 (PRD)

## 1. 프로젝트 개요

| 항목 | 내용 |
|---|---|
| **프로젝트명** | BEBE PET 공식 웹사이트 |
| **클라이언트** | 베베펫코리아 |
| **목적** | 천안 본점 매장 안내 + 분양/진료/사료/미용 서비스 소개 + 방문 예약 접수 |
| **타겟 디바이스** | Desktop (1280px 이상) 우선, 모바일 (375px 이상) 반응형 지원 |
| **디자인 톤** | 어두운 시네마틱 풀스크린 + 서브틀한 인터랙션 |
| **언어** | 한국어 / 영어 (사용자 토글) |

## 2. 핵심 비즈니스 목표

1. **방문 예약 유도** — 1차 전환 목표. 우측 상단 RESERVATION CTA + 예약 모달
2. **브랜드 인지** — 인트로 타이핑 → 풀스크린 영상 → 브랜드 카피로 톤 전달
3. **카테고리 안내** — 분양 / 병원 / 사료·간식 / 용품 / 미용·스파 5개 서비스
4. **위치 명확 전달** — 천안 본점 위치 시각화 (카카오맵 임베드)
5. **Korean PIPA 컴플라이언스** — 개인정보 수집 동의 + 처리방침 페이지

## 3. 페이지 구조

```
/ (홈)
├── ScrollHero (인트로 → stage 1/2/3)
├── CategoryIndex (5개 카테고리 hover preview)
├── Contact / VISIT (지도 + 매장 정보)
└── Footer (브랜드 + 사업자정보 + 개인정보처리방침 링크)

/news (공지사항 목록)
/privacy (개인정보처리방침)
```

## 4. 기능 요구사항 (요약)

자세한 사양은 [03_기능명세서.md](./03_기능명세서.md) 참고.

### 구현 기능 (본 산출물 범위)

- Hero 인트로 타이핑 애니메이션 및 Skip 버튼
- Hero 3-stage 비디오 frame morph (vis.mp4 / brand.mp4)
- Top nav (브랜드 / 분양 / 병원 / 사료&간식 / 용품 / 미용 / 공지사항 / 방문예약등록)
- Top nav 클릭 시 해당 섹션 부드러운 스크롤 — 카테고리 항목은 `#index`의 hover preview 활성화
- CategoryIndex 마퀴 hover → 카테고리별 프리뷰 (사진 / 라벨 / 본문 / CTA), 스크롤 중 hover 잠금
- Visit 섹션 카카오맵 정적 이미지 임베드 + 베베펫 본사 오버레이 마커
- 방문 예약 모달 (이름 / 연락처 / 지역 / 상담목적 / 개인정보 동의) — 클라이언트 단 검증 후 alert 응답
- KOR / ENG 언어 토글 (`localStorage` 영속화, 모든 사용자 노출 텍스트 i18n 처리)
- 모바일 햄버거 메뉴
- 반응형 레이아웃 (`max-md` / `md` 분기)
- 개인정보처리방침 페이지 (`/privacy`) — KISA 표준 9개 섹션
- 공지사항 페이지 (`/news`) — 정적 더미 데이터

## 5. 비기능 요구사항

| 항목 | 요구치 |
|---|---|
| **성능** | LCP < 2.5s, FID < 100ms (Lighthouse 기준) |
| **접근성** | WCAG 2.1 AA 수준 (alt 텍스트, 키보드 네비, aria-label) |
| **SEO** | 페이지별 metadata, OpenGraph (선택) |
| **호환성** | Chrome / Edge / Safari / Firefox 최신 2버전, iOS Safari, Android Chrome |
| **언어** | UTF-8, html lang 자동 전환 (KO ↔ EN) |
| **보안** | 외부 링크 `rel="noopener noreferrer"`, 시크릿 노출 0건 |
| **법규 준수** | 개인정보 보호법 (PIPA), 전자상거래법 (사업자 정보 footer 노출) |
