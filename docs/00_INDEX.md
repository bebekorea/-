## 추가 자산

- `/privacy` 라우트: 사이트 내 개인정보처리방침 페이지. KISA 표준 9개 섹션 구성. 사업자 실제 정보 입력 및 법무 검토 후 운영 권장.
- 소스 코드: `/src`, `/public`, `/scripts` 등 프로젝트 루트 일체
